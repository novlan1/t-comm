
## `clipboardMp(text)` 


**描述**：<p>小程序粘贴</p>

**参数**：


| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| text | <code>string</code> | <p>待复制的文本</p> |



**示例**

```ts
clipboardMp('stupid').then(() => {});
```
