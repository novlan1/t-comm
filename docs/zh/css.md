## 引入方式

```ts
import { removeCss } from 't-comm';

// or

import { removeCss} from 't-comm/lib/css/index';
```


## `removeCss(href)` 


**描述**：<p>移除CSS</p>

**参数**：


| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| href | <code>string</code> | <p>CSS链接</p> |



**示例**

```typescript
removeCss('https://xxx.css')
```
