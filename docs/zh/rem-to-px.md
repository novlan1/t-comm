## 引入方式

```ts
import { remToPxInFile } from 't-comm';

// or

import { remToPxInFile} from 't-comm/lib/rem-to-px/index';
```


## `remToPxInFile(filePath)` 


**描述**：<p>替换文件的 rem 单位，转为 px</p>

**参数**：


| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| filePath | <code>string</code> | <p>文件路径</p> |



**示例**

```ts
remToPxInFile('xxx.vue');
```
