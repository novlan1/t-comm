<h2>引入</h2>

```ts
import { saveMpImage } from 't-comm';

// or
import { saveMpImage} from 't-comm/lib/mp/index';
```


## `saveMpImage(url, options)` 


**描述**：<p>小程序下保存图片</p>

**参数**：


| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| url | <code>string</code> | <p>图片地址</p> |
| options | <code>object</code> | <p>提示选项</p> |



**示例**

```ts
saveMpImage('https://xxx.png');
```
